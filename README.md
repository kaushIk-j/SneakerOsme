
# SneakerOsme

E-commerce website for a sneaker brand "SneakerOsme" built in ReactJs. 


## Tech Stack

- Frontend: React.js, HTML, CSS
- Backend: MockBee
- Payment Integration: Razorpay
- Deployment: Netlify
- Other Libraries and Tools: React Router, Axios, JWT, etc.

## Features

- Authentication
  - Login
  - Logout
  - Signup
- Product Listing
- Filter Products by
  - Category
  - Discount
  - Price
  - Rating
- Cart Management
- Wishlist Management
- Search by
  - Product Name
  - Category
- Address Management
- Single Product Page
- Loading & Alerts
- User Profile Page
- Checkout
- Order Summary
- Order History
- Apply Coupons
- Payment Integration
- Responsive



## Run Locally

Clone the project

```bash
  git clone https://github.com/kaushIk-j/SneakerOsme.git
```

Go to the project directory

```bash
  cd react-sneakers-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
